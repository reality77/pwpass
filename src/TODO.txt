**** APPELS REST
PasswordController : alias 'p'
- GET(key) : récupère un mot de passe stocké pour la clé key 
- POST(key) : génère et stocke un mot de passe  pour la clé key
- POST(key, pwd) : stocke le mot de passe pwd pour la clé key
- PUT(key) : génère un nouveau mot de passe pwd pour la clé key
- PUT(key, pwd) : change le mot de passe pwd pour la clé key
- DELETE(key) : supprimme le mot de passe pour la clé key
